# もふ梟の迷宮狩り

GitHub Pagesで公開できる、スマホ向けターン制ハクスラの最小プロトタイプです。

## 遊べる内容

- スワイプまたは方向ボタンで1マス移動
- 敵の方向へ進むと攻撃
- 1回行動すると敵も行動
- 速い敵は2歩、遅い敵は2ターンに1歩
- 敵を倒すとEXPが入り、レベルアップします
- 装備、薬草、宝箱、戦利品が落ちます
- 敵を全滅させると階段が出て、次の階層へ進めます
- 3階ごとにボスが出ます
- 「羽根斬り」で前方3マスまで攻撃できます
- 「薬草」でHPを回復できます
- レトロ調BGMと行動に応じた効果音が鳴ります
- 音量とミュート設定は次回も引き継がれます
