# もふ梟の迷宮狩り

GitHub Pagesで公開できる、スマホ向けターン制ハクスラの最小プロトタイプです。

## 遊べる内容

- スワイプまたは方向ボタンで1マス移動
- 敵の方向へ進むと攻撃
- 1回行動すると敵も行動
- 速い敵は2歩、遅い敵は2ターンに1歩
- 敵を倒すとEXPが入り、レベルアップします
- 装備、薬草、宝箱、戦利品が落ちます
- 敵を全滅させると階段が出て、次の階層へ進めます
- 3階ごとにボスが出ます
- 「羽根斬り」で前方3マスまで攻撃できます
- 「薬草」でHPを回復できます

## GitHub Pagesへの置き方

1. このフォルダの中身をリポジトリに入れる
2. GitHubのSettingsからPagesを開く
3. Branchを選び、公開フォルダを `/root` または `/docs` に設定する
4. 表示されたURLをスマホで開く

このまま静的ファイルだけで動きます。
